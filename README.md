## Scrollbar React

A simple scrollbar component for React.

## Installation
```bash
npm install @darwish/scrollbar-react
```

## Usage
```jsx
import Scrollbar from '@darwish/scrollbar-react';

function App() {
  return (
    <div>
      <Scrollbar>
        <div>
          <p>Some content here</p>
        </div>
      </Scrollbar>
    </div>
  );
}
```
